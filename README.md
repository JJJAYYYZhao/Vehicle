### 目标实现功能
以下主要功能：
1.  注册，登陆，个人信息
2.  可预约加油
3.  地图实时显示当前汽车位置，并显示周围的加油站
4.  根据路况选择路线
5.  维护车辆信息
6.  开车时可听音乐
7.  交通违章信息
